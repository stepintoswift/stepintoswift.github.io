## Typealias

A typealias is simply an alternate name for an existing type.

e.g: Giving a basic type more meaning: <br />
```
typealias ConfirmationMessageShown = Bool
```

e.g: Also useful for closures: <br />
```
typealias DismissHandler = () -> Void
```
